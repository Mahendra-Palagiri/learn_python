### Examples

#### Example 1 — Create and Inspect Tensors

```python
import torch

x = torch.tensor([[1.0, 2.0], [3.0, 4.0]])
print(x)
print(x.shape)
print(x.dtype)
```

#### Example 2 — Autograd on a Tiny Scalar Example

```python
import torch

x = torch.tensor(3.0, requires_grad=True)
y = x**2 + 2*x

y.backward()
print(x.grad)
```

#### Example 3 — A Minimal `nn.Module`

```python
import torch
from torch import nn

class TinyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.layer1 = nn.Linear(2, 4)
        self.relu = nn.ReLU()
        self.layer2 = nn.Linear(4, 1)

    def forward(self, x):
        x = self.layer1(x)
        x = self.relu(x)
        x = self.layer2(x)
        return x
```

#### Example 4 — Minimal Batch Training Skeleton

```python
for X_batch, y_batch in train_loader:
    optimizer.zero_grad()
    y_pred = model(X_batch)
    loss = loss_fn(y_pred, y_batch)
    loss.backward()
    optimizer.step()
```
